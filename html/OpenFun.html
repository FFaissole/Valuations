OpenFun : operator which take an open and return

the characteristic function of this open
such a construction is possible because of the way we have defined the lower reals as maps from Q to Sier. We define the OpenFun by induction on the Sierpinski sets

Section S_to_RlP.

Map from Sier to RlowPos
Definition S_to_RlP : ∀ s : Sier, RlowPos.

Monotonicity of the map from Sier to RlowPos
Lemma S_to_RlP_mon : ∀ s1 s2, s1 ≤ s2
                        → S_to_RlP s1 ≤ S_to_RlP s2.

S_to_RlP preserves meets
Lemma meet_pres_S_to_RlP :
          MeetPreserving (fun (s: Sier) ⇒ S_to_RlP s).

S_to_RlP preserves joins
Lemma join_pres_S_to_RlP :
          JoinPreserving (fun (s: Sier) ⇒ S_to_RlP s).

Coercion S_to_RlP : Sier >-> RlowPos.

End S_to_RlP.

Section OpenFun.

Map from Opens to characteristic function
Definition OpenFun (A : hSet) : ∀ (U : A → Sier),
                                       (A → RlowPos).

OpenFun is definite
Lemma OpenFun_def {A} : ∀ U:OS A, U = OS_empty
                               → OpenFun _ U = fun x ⇒ RlP_0.

OpenFun is sub-probabilistic
Lemma OpenFun_prob {A} : ∀ U:OS A, U = OS_full
                               → OpenFun _ U ≤ fun x ⇒ RlP_1.

OpenFun is monotonic
Lemma OpenFun_mon {A} : ∀ U V:OS A, U ≤ V → OpenFun _ U ≤ OpenFun _ V.

1_U (x) = 1 iff 1_U (x) > 0
Lemma OpenFun_0_1 {A : hSet}: ∀ (U : OS A) z,
    OpenFun A U z = RlP_1 ↔ RlP_0 < OpenFun A U z.

1_U (x) = 1 iff x ∈ U
Lemma OpenFun_correct {A : hSet} : ∀ (x:A) U,
               OpenFun A U x = RlP_1 ↔ (U x) = SierTop.

1(U /\ V) = (1_U) /\ (1_V)
Lemma Meet_pres_openfun {A:hSet}: ∀ z s,
          MeetPreserving (fun (U: OS A) ⇒ val (OpenFun A U z) s).

1(U \/ V) = (1_U) \/ (1_V)
Lemma Join_pres_openfun {A:hSet}: ∀ z s,
          JoinPreserving (fun (U: OS A) ⇒ val (OpenFun A U z) s).

1_U + 1_V = 1(U \/ V) + 1(U /\ V)
Lemma OpenFun_mod {A}: ∀ (U V : OS A), fplus (OpenFun _ U) (OpenFun _ V) =
                                fplus (OpenFun _ (OS_meet U V))
                                      (OpenFun _ (OS_join U V)).

Lemma D_op_OpenFun {A : hSet}: ∀ (U : OS A),
           (D_op 0 (OpenFun A U)) = U.

End OpenFun.
